# Domek Wonen v6

Modulo tecnico inicial para normalizar el seed CSV del harvester NVM.

## Requisitos

- Python 3.11+

## Comandos

Instalar dependencias:

```powershell
python -m pip install -r requirements.txt
```

Correr tests:

```powershell
python -m pytest
```

Ejecutar importador:

```powershell
python scripts/import_sources.py
```
