# Codex Task Planning

Cuando usarla:
Usa esta skill para redactar prompts pequenos y ejecutables para Codex.

Que archivos leer:
- `AGENTS.md`
- `docs/09_CODEX_WORKFLOW.md`
- docs del area objetivo

Objetivo:
Convertir objetivos amplios en tareas acotadas con validacion clara.

Workflow:
1. Define una sola fase o modulo por prompt.
2. Lista archivos exactos a leer.
3. Declara constraints y out-of-scope.
4. Define acceptance criteria y validacion real.

Output esperado:
- prompt pequeno y preciso
- criterios de cierre verificables

Reglas:
- Evitar prompts vagos o multi-fase.

Errores comunes:
- pedir arquitectura, runtime y dashboard en el mismo prompt
- no definir archivos de lectura
